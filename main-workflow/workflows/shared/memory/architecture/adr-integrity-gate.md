# ADR Integrity Gate

## Mandatory Behaviour
Before proposing or modifying architecture, the AI must:

1. Identify relevant ADRs
2. State compliance status:
   - Compliant
   - Extending
   - Violating

## ADR Extension Rules
If an ADR must be extended or violated:
- AI must explain why
- AI must propose a draft ADR update
- AI must not proceed without human approval

## Prohibited Behaviour
- Ignoring ADRs
- Assuming ADRs are outdated
- Introducing alternative patterns silently
